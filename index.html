Email Server & Ambient Display

I wrote scripts to automate an email account (send mail, read mail, and move messages to folders). Additional scripts scraped various websites for event information and stored it in a SQL database on a Raspberry Pi. Each week, users who have opted in receive a personalized email containing information about upcoming events from multiple sources based on categories they have selected. The Raspberry Pi was mounted on a customized disco ball so that database activity was reflected in the pattern of lights and motion of the disco ball. As an ambient display, the goal is to visualize digital information in a new way.

Pencil Code for Educators

Pencil Code is a block-based programming language designed to teach computer science and algortihmic thinking. Working with Google, I led a small team that built a prototype of a website for educators to share resources and gain inspiration to teach using Pencil Code. I conducted UX research to guide the site through several iterations.

Tangible Programming Environment

Using the shape displays at MIT's Tangible Media Group, I worked with a small team to develop a physical programming language that used gestures and physical interaction to perform basic commands. The goal was to take advantage of the benefits of tangibility to make learning programming more physical. Our team also produced a concept video.

Machine Learning Shakespeare Remix

I wrote a machine-learning algorithm that generated text that read and felt like Shakespeare, although it didn’t necessarily make sense. Inspired by Brendan Kohler's code, my algorithm read in Shakespeare’s collected works, then using Markov chains, picked words based on which word was most likely to occur next. The work was presented in class in response to Walter Benjamin’s “The Work of Art in the Age of Mechanical Reproduction.”

ExoMuscle

Whereas muscles convey information about the immediate physical world through kinesthetic feedback, ExoMuscle can convey information about the non-physical (digital) world beyond the reach of your immediate space. I was a member of the team that developed the concept, created a prototype, and proposed three applications: sensing driving mistakes, syncing with a calender to let users feel the "pressure" of time, and giving wearers a sixth sense by letting users feel radiofrequencies.
